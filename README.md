# bolot
bot do discord que usa inferência para identificar  diferentes tipos de bolo que o usuario mandar 🎂

🤓👆 Esse bot não possui uma utilidade real, ao contrario de bots de música ou de moderação! Ele foi criado com a intenção de ser algo inesperado e engraçado, já que "identificação de bolos" não é a primeira coisa que vem em mente quando se fala de bots de discord 

⚠️ O modelo usado vem do Teachable Machine, então tenha  certeza de instalar a versão adequada do python e tensorflow ⚠️
